# How to run this project locally

#### 1. Clone the Project
```
git clone https://github.com/yoshan0921/movies-app.git
```
#### 2. Navigate to the Project Directory
```
cd movies-app
```
#### 3. Install Dependencies
```
npm install
```
#### 4. Start the Development Server
```
npm start
```
